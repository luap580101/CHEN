## 已經npm i 的套件
- 框架: npm i express
- 啟動: npm i -g nodemon
- 資料庫: npm i mysql2
- 建立環境變數: npm i dotenv
- 跨源存取: npm i cors
- 驗證用: npm i express-validator
- 上傳圖片用: npm i multer
- session: npm i express-session session-file-store

### 開啟檔案請
- 建立.env檔案(請參考.env.example)
- npm i 更新node_modules
- 開啟後端指令: npm run dev
